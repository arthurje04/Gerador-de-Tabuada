
Objetivos 
• Compreender e aplicar as estruturas de repetição com a finalidade de 
familiarizar os estudantes com a estrutura for em JavaScript. 

• Aplicar conceitos teóricos de programação (variáveis, loops e operações 
matemáticas simples) em um contexto prático e relevante. 

Solução Digital 


• Ferramenta utilizada: VS Code (editor de código) 

• Tecnologias: HTML, CSS e JavaScript 

• Execução: Página web interativa que gera tabuadas de multiplicação. 

Procedimento / Atividade 


Atividade proposta:
Criar uma página web que gera tabuadas de multiplicação de 
forma interativa. 


Etapas: 
1. Abertura da Página: 
a. Exibir título “Gerador de Tabuada Interativo”. 
b. Campo de entrada para o usuário digitar um número entre 1 e 10. 
2. Inserção de Dados: 
a. Usuário insere o número desejado. 
3. Cálculo e Exibição: 
a. Usar a estrutura de repetição for para calcular e exibir a tabuada do 
número escolhido (1 a 10). 
4. Exploração e Interatividade: 
a. Usuário pode inserir diferentes números e visualizar novas tabuadas 
dinamicamente.
